Rakshane

Rakshane is the ALL-IN-ONE safety device which you'll need to protect yourself. The purpose of this device is to safeguard everyone in the event they might face any danger. Other safety devices available in the market are either SOS sending devices or protection devices but Rakshane on the other hand not only sends a SOS signal to the important people but it also sends the user location using the google maps. The main advantage of this project is that this device can be carried everywhere since it is small.
